# 전용 서버 컴퓨터 세팅 가이드 (윈도우)

새로 준비하신 **윈도우 컴퓨터(서버)**에 이 프로그램을 설치하고 실행하는 방법입니다.
같은 공유기(와이파이)를 쓰는 모든 직원이 접속할 수 있습니다.

## 1. 윈도우용 프로그램(Node.js) 설치
새 컴퓨터에서 아래 작업을 수행해주세요.
1.  [Node.js 공식 홈페이지](https://nodejs.org/ko) 접속.
2.  **LTS 버전** (안정적 버전) 다운로드 및 설치.
3.  설치 중 나오는 체크박스는 모두 기본값으로 두고 'Next'만 누르시면 됩니다.

## 2. 프로젝트 파일 옮기기
1.  현재 작업하던 컴퓨터의 `second-hand-inventory` 폴더를 통째로 복사합니다. (USB나 메일 등 이용)
    -   *팁: `node_modules` 폴더와 `.next` 폴더는 용량이 크니 제외하고 복사하셔도 됩니다.*
2.  새 컴퓨터의 원하는 위치(예: `C:\inventory`)에 붙여넣습니다.

## 3. 프로그램 실행 준비
새 컴퓨터에서 다음을 실행합니다.
1.  **폴더 열기**: 방금 복사한 폴더로 들어갑니다.
2.  **터미널 열기**: 폴더 빈 공간에 `Shift + 우클릭` -> **"PowerShell 창 열기"** (또는 "터미널에서 열기") 클릭.
3.  **설치 명령어 입력**: (복사할 때 `node_modules`를 제외한 경우 필수)
    ```powershell
    npm install
    ```
4.  **데이터베이스 초기화**: (만약 새 PC에서 완전히 새로 시작하려면)
    ```powershell
    npm run migrate
    ```
    *(기존 데이터를 그대로 쓰고 싶으시면 `inventory.db` 파일을 꼭 같이 복사해오셔야 합니다.)*

## 4. 서버 실행
터미널에 다음 명령어를 입력하면 서버가 켜집니다.
```powershell
npm run dev:lan
```

## 5. 접속 주소 확인
새 컴퓨터의 터미널에서 `ipconfig`를 쳐서 **IPv4 주소**를 확인하세요. (예: `192.168.0.50`)
직원들에게 알려줄 주소는 다음과 같습니다:
👉 **http://192.168.0.50:3000**

## [중요] 방화벽 경고가 뜨면?
서버를 처음 켤 때 윈도우 보안 경고창이 뜹니다. 반드시 **"홈 네트워크, 개인 네트워크"** 항목을 체크하고 **"허용"**을 눌러주세요. 그렇지 않으면 다른 직원이 접속할 수 없습니다.
