# MD-SOGAE v2.9 Workflow Diagram

```
┌─────────────────────────────────────────────────────────────────┐
│                    MD-SOGAE v2.9 Protocol                       │
│          패션 아카이브 전문가 및 자산 평가사 시스템                 │
└─────────────────────────────────────────────────────────────────┘
                                 │
                                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  Phase 1: Visual & OCR Priority (데이터 채굴)                    │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                 │
│  📸 이미지 입력                                                  │
│      │                                                          │
│      ├─► 🏷️  케어라벨 스캔 (OCR)                               │
│      │    └─► Product Code (Art No., Style No., RN)           │
│      │                                                          │
│      ├─► 🧵 소재 분석                                           │
│      │    └─► Fabric Composition (Nylon 100%, Cotton 80%)     │
│      │                                                          │
│      ├─► 🏢 브랜드/라인 판별                                     │
│      │    └─► Brand + Sub-Line (Prada Sport, Shadow Project)  │
│      │                                                          │
│      └─► 📏 사이즈 & 등급                                       │
│           └─► Size (M, L, 95) + Grade (S, A, B)               │
│                                                                 │
│  Output: CareLabel { productCode, fabric, brand, size, ... }   │
└─────────────────────────────────────────────────────────────────┘
                                 │
                                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  Phase 2: Market Intelligence (가격 산출)                        │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                 │
│  🌍 글로벌 인덱스 (Global Anchor)                               │
│      ├─► eBay Sold Listings                                    │
│      └─► Grailed Sold Items                                    │
│           └─► ⚠️ 관세 가중치(1.18x) 절대 적용 금지              │
│                                                                 │
│  🇰🇷 국내 시장 스캔 (Local Real)                                │
│      ├─► KREAM (실거래 체결가) ⭐ 가장 강력한 기준점             │
│      ├─► 무신사 USED (유사 등급 판매가)                         │
│      ├─► 번개장터 (실시간 매물 호가)                            │
│      └─► 후르츠패밀리 (전문 셀러 리스팅가)                       │
│                                                                 │
│  💰 최종 가격 결정                                              │
│      └─► 글로벌 + 국내 4대 플랫폼 교차 검증                     │
│           └─► 즉시 판매 가능가 산출                            │
│                                                                 │
│  Output: MarketPrice { global, kream, used, final, reason }    │
└─────────────────────────────────────────────────────────────────┘
                                 │
                                 ▼
┌─────────────────────────────────────────────────────────────────┐
│           🔍 Metadata Card (중간 검수창)                         │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                 │
│  ┌───────────────────────────────────────────────────────┐    │
│  │  ✅ 추출 품번: ABC-12345                               │    │
│  │  ✅ 판별 소재: Nylon 100%                              │    │
│  │  ✅ 산출 가격: ₩165,000                                │    │
│  │  ✅ 추천 제목: [Technical] Stone Island...            │    │
│  └───────────────────────────────────────────────────────┘    │
│                                                                 │
│  [승인 및 상세 정보 보기]  [수정]                                │
└─────────────────────────────────────────────────────────────────┘
                                 │
                                 ▼ (승인 후)
┌─────────────────────────────────────────────────────────────────┐
│  Phase 3: Professional Naming (50자 이내)                       │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                 │
│  📝 구조: [전문태그] 브랜드 연식+모델명 (특징/핏) 성별-사이즈     │
│                                                                 │
│  전문 태그:                                                     │
│      ├─► [Technical] - 기능성 소재 중심                        │
│      ├─► [Archive] - 역사적 가치 빈티지                        │
│      ├─► [Sartorial] - 테일러링/코트류                         │
│      └─► [Original] - 브랜드 시그니처                          │
│                                                                 │
│  성별-사이즈: MAN-L, WOMAN-M, KIDS-150, UNISEX-F               │
│                                                                 │
│  예시:                                                          │
│  [Technical] Stone Island 23FW Shadow Project 고어텍스 재킷 MAN-L │
│  [Archive] Helmut Lang 1998 본디지 카고팬츠 UNISEX-M           │
│                                                                 │
│  Output: ProfessionalName { fullName, tag, genderSize, ... }   │
└─────────────────────────────────────────────────────────────────┘
                                 │
                                 ▼
┌─────────────────────────────────────────────────────────────────┐
│  Phase 4: Editorial Content (전문가적 상세페이지)                 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                 │
│  📖 Brand Heritage (브랜드 헤리티지)                            │
│      └─► 모델의 역사적 맥락 + 브랜드 철학 (2-3문장)             │
│                                                                 │
│  🔬 Detail Guide (디테일 가이드)                                │
│      └─► 소재 특성 + 부자재 분석 + 품질 평가 (3-4문장)          │
│                                                                 │
│  💎 Archive Value (아카이브 가치)                               │
│      └─► 시세 데이터 기반 구매 당위성 + 희소성 (2-3문장)        │
│                                                                 │
│  Output: EditorialContent { heritage, detail, value }          │
└─────────────────────────────────────────────────────────────────┘
                                 │
                                 ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ✅ 상품에 적용하기                            │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                                 │
│  모든 분석 결과를 상품 정보에 자동 입력:                          │
│      ✓ 상품명 (Professional Name)                              │
│      ✓ 품번 (Product Code)                                     │
│      ✓ 소재 (Fabric Composition)                               │
│      ✓ 판매가 (Final Price)                                    │
│      ✓ 상세설명 (Editorial Content)                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⏱️  Total Processing Time: ~60 seconds
🎯  Accuracy: 85-90% (품번/소재), 80% (가격)
🔑  Key Feature: 관세 가중치 배제 + 실거래가 기반 가격 산출
```
