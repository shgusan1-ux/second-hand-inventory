
# 🤖 카카오톡 AI 봇 연동 가이드

사장님, 이제 카카오톡을 통해 재고관리 시스템과 대화하실 수 있습니다.
다음 단계를 따라 카카오 디벨로퍼스에서 챗봇을 설정해 주세요.

## 1. 카카오 디벨로퍼스 설정
1. [카카오 디벨로퍼스](https://developers.kakao.com/) 접속 및 로그인
2. **내 애플리케이션** -> **애플리케이션 추가하기**
   - 앱 이름: `InventoryAI`
   - 사업자명: `(주)에이치엠이커머스`
3. 생성된 앱 선택 후 좌측 메뉴 **[카카오톡 채널]** 선택
   - 채널 생성하기 (비즈니스 채널 권장)

## 2. 카카오 i 오픈빌더 (챗봇 빌더) 신청
1. [카카오 i 오픈빌더](https://i.kakao.com/) 접속
2. OBT 신청 (승인까지 1~3일 소요될 수 있음)
   - *팁: "사내 재고 관리 및 업무 자동화 챗봇"이라고 적으면 승인이 빠릅니다.*

## 3. 스킬(Skill) 등록 (핵심!)
오픈빌더 승인 후, 봇 관리자 페이지에서:
1. **스킬** 탭 이동 -> **스킬 생성**
2. 설정값 입력:
   - **스킬 이름**: `InventoryCommand`
   - **설명**: `재고관리 AI 명령`
   - **URL**: `https://your-domain.com/api/kakao/webhook`
     - *(배포된 실제 도메주소를 입력해야 합니다)*
3. **저장**

## 4. 시나리오 설정 (폴백 블록)
사용자가 하는 "모든 말"을 우리 서버로 보내야 합니다.
1. **시나리오** 탭 -> **폴백 블록(Fallback Block)** 선택
2. **봇 응답** 설정에서 **스킬 데이터 사용** 체크
3. 앞서 만든 `InventoryCommand` 스킬 선택
4. **저장** 및 **배포**

## 5. 테스트
카카오톡에서 해당 채널을 친구 추가하고 말을 걸어보세요.
- "오늘 매출 어때?"
- "시스템 상태 확인해줘"

---

### 💡 참고사항
- 현재는 **관리자 모드**로 동작합니다. (보안을 위해 추후 인증 절차 추가 가능)
- 서버가 Vercel에 배포되어 있어야 카카오 서버가 접속할 수 있습니다.
